# Pixel Telo - 应用发布

> 专为 Google Pixel 设备设计的来电识别与拦截应用，轻量级、隐私优先、零打扰的原生体验。

---

## 📦 下载安装包

前往 **[Releases 页面](https://github.com/PixelTelo/AppRelease/releases/latest)** 下载最新版本。

| 文件 | 说明 |
|---|---|
| `PixelTelo-vX.X.apk` | Universal APK，直接安装到手机 |
| `PixelTelo-vX.X.aab` | Android App Bundle，供开发者使用 |

**安装方式**：下载 `.apk` 文件后，在手机文件管理器中打开即可安装（需开启「允许未知来源」）。

## 🔒 隐私说明

Pixel Telo 隐私优先：不收集任何个人数据，所有来电识别均在本地设备处理。

---

**兼容性**：Android 10（API 29）及以上，推荐 Google Pixel 系列设备。
